Coffee Service App
Простий та зручний додаток для обліку ремонту кавоварок, створений для майстрів сервісних центрів.

📋 Створення актів ремонту

🔢 Швидке додавання робіт по коду

🧾 Автоматичний підрахунок суми

🏷 Збереження моделі та серійного номера

📜 Історія ремонтів для кожної кавоварки

🛠 Відображення виконаних робіт у кожному ремонті

❌ Видалення окремих ремонтів

💰 Облік заробітку

💸 Облік виплат

🧮 Автоматичний розрахунок залишку

📊 Графік заробітку

📱 Адаптація під мобільні пристрої

🎨 Сучасний UI з плаваючими кнопками та popup-вікнами

📊 Як це працює
Вводиш модель та серійний номер кавоварки

Додаєш роботи через коди або вручну

Формується акт ремонту

Дані зберігаються в історію

Система автоматично рахує: 

   загальний заробіток
 
   виплати
  
   залишок

У будь-який момент можна переглянути: 

  історію конкретної кавоварки
  
  список виконаних робіт
  
  графік заробітку

💾 Збереження даних

Усі дані зберігаються локально в браузері через localStorage.

⚠️ Важливо: при очищенні браузера дані можуть бути втрачені.

📦 Запуск

Просто відкрий файл:
index.html 

Нічого встановлювати не потрібно.
